THREATS

Five critical OS-level vulnerability classes, each with distinct attack vectors, detection signatures, and mitigation strategies.

CVE CLASS 001
⚡
BUFFER OVERFLOW

Overwrites adjacent memory by exceeding allocated buffer bounds, enabling control-flow hijacking via corrupted return addresses or function pointers.

CRITICAL — CVSS 9.8
CVE CLASS 002
🔓
TRAPDOOR / BACKDOOR

Hidden authentication bypass mechanisms embedded in software or firmware, enabling persistent unauthorised access without triggering standard auth flows.

CRITICAL — CVSS 9.1
CVE CLASS 003
🌐
CACHE POISONING

Injects malicious entries into DNS or web caches by exploiting predictable transaction IDs, redirecting users to attacker-controlled infrastructure.

HIGH — CVSS 8.2
CVE CLASS 004
⛓
ROP CHAIN

Return-Oriented Programming chains existing code gadgets to build arbitrary computation without injecting new code, defeating DEP and NX protections entirely.

CRITICAL — CVSS 9.3
CVE CLASS 005
⏱
RACE CONDITION (TOCTOU)

Exploits the window between a security check and resource use, allowing attackers to swap checked resources before the operation completes — bypassing privilege checks.

HIGH — CVSS 7.8
COMING SOON
+
MORE VECTORS

Heap spraying, use-after-free, format string attacks, integer overflow, and privilege escalation modules in development.
